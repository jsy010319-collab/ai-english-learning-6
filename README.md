# AI 영어 회화 앱

시나리오 기반 대화로 영어를 연습하고, AI가 즉시 교정과 팁을 제공하는 최소 구현(full‑stack) 웹 앱입니다. UI와 안내는 한국어, 학습 대화는 영어를 기본으로 합니다.

## 주요 기능
- 문화/카테고리별 시나리오 선택(예: 식당, 택시, 한국 카페)
- 실제 대화처럼 진행: AI가 짧게 질문, 사용자가 음성/텍스트로 답변
- 매 턴 교정과 한 줄 팁 제공(간결한 JSON 응답)
- 대본(스크립트) 형식의 대화 기록
- 어려운 문장 저장 및 간격 반복 복습(클라이언트 로컬 저장)
- 깔끔하고 접근성 고려한 UI, 라이트/다크 대응

## 빠른 시작
1. Node.js 18+와 npm을 설치하세요.
2. 프로젝트 루트에 `.env` 파일을 만들고 아래를 입력하세요:
   ```
   OPENAI_API_KEY=YOUR_OPENAI_KEY_HERE
   PORT=3000
   ```
3. 설치 및 실행:
   ```bash
   npm install
   npm start
   ```
4. 브라우저에서 `http://localhost:3000` 접속.

## 참고
- OpenAI `gpt-4o-mini` Chat Completions와 JSON 모드를 사용해 간결한 피드백을 받습니다.
- 음성 입/출력은 브라우저 Web Speech API를 사용하며, 미지원 환경에서는 텍스트 입력으로 대체됩니다.
- API 키는 서버의 환경변수로만 사용되며, 프론트엔드에 포함하지 않습니다.
- 음성 인식/합성 언어는 기본값이 영어(`en-US`)입니다. 한국어로 바꾸려면 `public/app.js`의 `recognition.lang` 또는 `SpeechSynthesisUtterance`의 `lang` 값을 `ko-KR`로 변경하세요.

## 안전
- API 키를 프론트엔드 파일에 하드코딩하지 마세요. 반드시 환경변수를 사용하세요.
